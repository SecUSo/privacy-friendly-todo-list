Приватный Список Дел может быть использован для сохранения текущих задач и организации личного расписания. Данное приложение позволяет создавать списки задач, которыми можно управлять индивидуально. Каждый список содержит в себе набор задач. Каждая задача может иметь крайний срок, время напоминания, и список подзадач. Путём установки времени напоминания, пользователь будет уведомлён через уведомление. Возможно отображать все задачи в календаре, защитить приложение с помощью секретного PIN-кода, а также сортировать и приоритизировать задачи в пределах списка. Цвета отражают срочность задачи (принимая во внимание крайний срок).

Функции Приватного Списка Дел

1. Защита PIN-кодом
Откройте меню настроек, чтобы установить PIN-код для защиты приложения. PIN-код должен содержать в себе как минимум 4 цифры. Он будет запрашиваться каждый раз после запуска приложения / открытия главного экрана.

2. Напоминание
Вы можете установить напоминание для ваших задач, которое оповестит вас о приближении крайнего срока. Настройки дают вам возможность включить звук уведомления.

3. Виджет
Дополнительно, данное приложение позволяет использовать виджет, который вы можете добавить на домашний экран своего устройства. Виджет показывает вам задачи, которые относятся к выбранному списку. Нажатие на задачу или её флажок приведёт к открытию главного экрана Приватного Списка Дел.

4. Прогресс через подзадачи
По умолчанию, прогресс может быть свободно установлен пользователем при создании или редактировании задачи. Список Дел даёт вам возможность автоматически высчитывать и отображать прогресс задачи, зависящий от выполенных подзадач. Вы можете зайти в настройки, и включить прогресс через подзадачи.

Наше Приватное Приложение отличается от других приложений по трём аспектам:

1) Минимальное количество разрешений
Приватный Список Дел использует только те разрешения, которые строго необходимы для его функциональности.

2) Отсутствие рекламы
В Приватном Списке Дел полностью отсутствует реклама.
Многие другие бесплатные приложения в Магазине Google Play показывают черезчур много назойливой рекламы, которая также быстрее разряжает аккумулятор.

3) Нет отслеживания
Приватный Список Дел не использует какие-либо трекеры.
